# PROJETO-MAGI
O Projeto MAGI, inspirado no anime Neon Genesis Evangelion, é um projeto experimental de robótica cujo objetivo é aprofundar conhecimentos em visão computacional, robótica e eletrônica.  Por meio deste site, serão registradas e documentadas todas as etapas de desenvolvimento, atualizações e evoluções do sistema ao longo do projeto.
